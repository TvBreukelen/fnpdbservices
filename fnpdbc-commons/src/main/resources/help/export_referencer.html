Export To P. Referencer

Convert (FNProg2PDA only)

  • Convert Images to Image files
    • By activating this checkbox all images to be exported are converted to bitmap files. This allows you to import these images in your P. Referencer database. To import the image filename (instead of the actual image), leave the checkbox deactivated.

    Important: the image conversion process takes a huge amount of processing time! You can't change the image size, because the Easy Referencer Tool requires images with a height of 100 and a width of 160 pixels.

    • Incl. Paragraph Header Activate this checkbox, if you'd like to include a header for the paragraph field.
    • Incl. Title Header You can assign a header to the title field by activating this checkbox. In that case FNProg2PDA "treats" the title field like any other field that follows.
    Examples:
    1. Incl. Paragraph Header = ON, Incl. Title Header = "OFF"
      Series: Harry Potter
      The Philosopher's Stone
      The Chamber of Secrets

    2. Incl. Paragraph Header = OFF, Incl. Title Header = "ON"
      Harry Potter
      Title: The Philosopher's Stone
      Title: The Chamber of Secrets

    3. Incl. Paragraph Header = ON, Incl. Title Header = "ON"
      Series: Harry Potter
      Title: The Philosopher's Stone
      Title: The Chamber of Secrets
    4. Incl. Paragraph Header = OFF, Incl. Title Header = "OFF"
      Harry Potter
      The Philosopher's Stone
      The Chamber of Secrets
