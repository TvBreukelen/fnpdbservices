Welcome

DBConvert is a freeware database x-converter tool that can convert a Calc (ODS), Dbase, Foxpro, JFile, Json, List, MobileDB, MS Access, MySql/MariaDB, PostgreSQL, MS Excel, Pilot-DB, Ascii delimited Textfile, VCard, XML or Yaml file to a Calc (ODS), Dbase (PC), Foxpro (PC), HanDBase (iPod/iPad/Palm/PocketPC), JFile (Palm), List (Palm), MobileBD (Palm/PocketPC), Pilot-DB (Palm), MS-Excel, SQLite, user definable text or XML file. This tool is included as well in the FNProg2PDA "suite".
