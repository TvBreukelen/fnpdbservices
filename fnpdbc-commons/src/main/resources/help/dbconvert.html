Profiles Tables

Here you can see al Profiles that are currently defined in DBConvert. Each export profile, that you defined is grouped alphabetically by export file type on tabs. This makes it easier to distinquish where the program exports to. in the table you can find some information about the profile itself. If you wish you could add some Notes as well.

Buttons

Button Description
Allows you to create a new profile
Edits the selected Export File Type or Profile
Makes an exact copy of the selected Profile
Removes the selected Export File Type or Profile
View or edit the records to be exported
Shows information about this screen
Exits the program
Export Starts the Export process

Checkbox

  • Deactivate Filter
    • By activating this option, DBConvert switches off a previously defined filter, thus exporting all records found in the import file.

    Menu Bar

    • File
      • Add New Profile
        • Creates a new export Profile
      • Backup
        • Creates a backup copy of all DBConvert Profiles
      • Restore
        • Restore DBConvert Profiles from a backup copy
      • Exit
        • Exits from DBConvert.
    • Config
      • General
        • In the General Configuration screen you can specify global settings for the date- and time format and conversion values for (un)selected checkboxes in the DBConvert software.
      • Encoding
        • If you are using non-latin characters like greek, cyrillic or chinese then you can here change the character set encoding so that DBConvert processes your characters correctly
      • Languages
        • Here you can select the DBConvert GUI-language (Default = English). This option is not visble is case DBConvert was started from the FNProg2PDA Tools menu.
      • Look and Feel
        • DBConvert uses by default the standard System "look and feel" (Like Windows on a MS-Windows PC). Here you can change the "look and feel" to the Java ("Metal") standard or any other "look and feel" that is available in the Java version that you've installed on your PC. This option is not visble is case DBConvert was started from the FNProg2PDA Tools menu.
    • Sites
      • Here you visit a couple of DBConvert related websites.

    • Help
      • The Help Menu provides you with some background information about the DBConvert program.
