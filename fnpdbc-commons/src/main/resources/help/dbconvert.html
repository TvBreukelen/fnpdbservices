Main Screen Help

Menu Bar

  • File
    • Add New Profile
      • Creates a new export Profile
    • Backup
      • Creates a backup copy of all DBConvert Profiles
    • Restore
      • Restore DBConvert Profiles from a backup copy
    • Exit
      • Exits from DBConvert.
  • Tools
    • Languages
      • Here you can select the DBConvert GUI-language (Default = English).
    • Look and Feel
      • DBConvert uses by default the standard System "look and feel" (Like Windows on a MS-Windows PC). Here you can change the "look and feel" to the Java ("Metal") standard or any other "look and feel" that is available in the Java version that you've installed on your PC.
    • Settings
      • In the General Settings screen you can specify global settings for the date-, time- and duration formats; the automated search for new versions and the default folder settings.
  • Sites
    • Here you visit a couple of DBConvert related websites.

  • Help
    • The Help Menu provides you with some background information about the DBConvert program.

Buttons

Profiles Tables

Here you can see all Profiles that are currently defined in DBConvert. Each export profile, that you defined is grouped alphabetically by export file type on tabs. This makes it easier to distinguish where the program exports to. in the table you can find some information about the profile itself. If you wish you could add some Notes as well.

Table

Column Description
Edit Edit the selected profile
Profile Name The Profile name (editable)
Export From The file type of the file we are exporting from
Table/Sheet The table or worksheet we are exporting from
Export From File The file name of the file we are exporting from
Export To File The file name of the file we are exporting to (editable)
Last Export Timestamp of the last export (editable)
Notes Optional notes (editable)

Checkbox

  • Deactivate Filter
    • By activating this option, DBConvert switches off a previously defined filter, thus exporting all records found in the import file.
