Export from a remote database

Field definition

Host name / IP

Connect to the database server on the given host. The default host is 127.0.0.1 (localhost). By default, MariaDB does not permit remote logins

Port

The TCP/IP port number to use for the connection. The default is 3306.

User

The database user name to use when connecting to the server.

Password

The password of the database account.

Database

The database on the server you'd like to connect to.
