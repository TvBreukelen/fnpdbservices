Export To JFile, MobileDB or Pilot-DB

Export Data

  • Into New Database
    • This option creates a new database. Should a previous copy of the "Export To" database file already exist then DBConvert/FNprog2PDA will overwrite it.
  • Into Current Database, Appending to existing records
    • Similar as the previous option, but in this case the new records are appended to the existing records in the database.

Convert

  • Booleans To Boolean Fields
    • Boolean values can be exported as booleans or as test. Should you leave this option deactivated then DBConvert/FNProg2PDA uses the replacement text values entered for "True" and "False" on the Conversion Tab of the General Configuration.
  • Dates To Date Fields
    • DBConvert/FNProg2PDA converts by default dates to text, using the date format settings specified in the General Configuration. If you activate this option then the dates are converted to JFile-, MobileDB- or Pilot-DB date Fields.
  • Time To Time Fields
    • DBConvert/FNProg2PDA converts by default times to text, using the time format settings specified in the General Configuration. If you activate this option then the time are converted to JFile-, MobileDB or Pilot-DB time Fields.
