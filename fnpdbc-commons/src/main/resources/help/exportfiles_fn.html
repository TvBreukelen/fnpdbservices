Export From

  • Combobox
    • In the combobox you can select the format of the "Export From" file (BookCAT, CATraxx, CATVids, SoftCAT or StampCAT).
  • Filename / Browse
    • You can use the Browse button to select the database to be exported
  • PDA Database Name
    • This field show the PDA database name associated to the "Export From" file. If left unchanged, FNProg2PDA uses this as default PDA database name for the export file. Should your export file require another name then you could enter the new name in this field.

Export To

  • Combobox
    • In the combobox you can select the format of the "Export To" file (Calc, HanDBase, JFile, List, MobileDB, MS-Excel, Textfile, DBase3, DBase4, Dbase5, FoxPro or Xml).
  • Filename / Open
    • You can use the Open button to select the database to export to or type in the name of the database directly in the text field.
  • Create Backup
    • Should the export file already exist then this option allows you to create a backup copy of the previous export file. If you deactivate the option then the previous export file will be deleted.
  • Encoding
    • Here you can override the default character set encoding for the "Export To" file. This feature can be very useful if the file to be exported will be used on another computer or OS that doesn't support the default characterset.

    Notes: FNProg2PDA verifies whether the export file exists and whether its format corresponds with the selected database format. DBConvert/FNProg2PDA is not able to write encrypted databases
