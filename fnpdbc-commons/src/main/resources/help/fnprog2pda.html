Main Screen Help

Menu Bar

  • File
    • Add New Profile
      • Creates a new export Profile
    • Backup
      • Creates a backup copy of FNProg2PDA settings
    • Restore
      • Restores the FNProg2PDA settings from a backup copy
    • Exit
      • Exits from FNProg2PDA.
  • Tools
    • Languages
      • Here you can select the FNProg2PDA GUI-language (Default = English).
    • Look and Feel
      • FNProg2PDA uses by default the standard System "look and feel" (Like Windows on a MS-Windows PC). Here you can change the "look and feel" to the Java ("Metal") standard or any other "look and feel" that is available in the Java version that you've installed on your PC.
    • Settings
      • In the General Settings screen you can specify global settings for the date-, time- and duration formats; the automated search for new versions and the default folder settings.
    • Check for update
      • This menu item checks if there is a later version of FNProg2PDA available for download.
  • Sites
    • Here you can visit a couple of FNProg2PDA related websites.

  • Help
    • The Help Menu provides you with some background information about the FNProg2PDA program.

Profiles Tables

Here you can see all Profiles that are currently defined in FNProg2PDA. Each export profile, that you defined is grouped alphabetically by export file type on tabs. This makes it easier to distinguish where the program exports to. in the table you can find some information about the profile itself. If you wish you could add some Notes as well.

Table

Column Description
Edit Edit the selected profile
Profile Name The Profile name (editable)
Export From The FNProgramvare database we are exporting from
Table/Sheet The table we are exporting from
Export From File The file name of the database we are exporting from
Export To File The file name of the file we are exporting to (editable)
Last Export Timestamp of the last export (editable)
Notes Optional notes (editable)

Checkboxes

  • New records Only
    • By activating this option FNProg2PDA will export only those records that were added to the FNProgramvare database since the previous Export.

  • New and modified records
    • Activate this checkbox, if you would like to export new and modified records only.

  • Deactivate Filter
    • By activating this option FNProg2PDA switches off a previously defined filter, thus exporting all records found in the import file.
